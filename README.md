### ts-tutorial
This tutorial is simple practice of typescript.
I have worked this project on VSCode so you might need to install some extensions.

### install NPM
``` C
npm install -g typescript
```

### tsconfig.ts
``` C
tsc --init
```

### VSCode Extensions
- CodeRunner
#### How to run ts code
1. go to .ts file you want to run
2. mouse right click
3. click 'Run Code'
